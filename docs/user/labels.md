# Guide utilisateur — Impression d'étiquettes

Les étiquettes produits permettent de **coller un code QR et un code-barres** sur chaque article. Cela facilite :
- Les **entrées et sorties de stock** (scanner = action instantanée)
- Les **ventes en caisse** (scanner le produit au lieu de le chercher)
- Les **inventaires** (pointer les articles scannés)

---

## Types d'étiquettes

### Étiquette thermique 58mm

Pour les **imprimantes thermiques Bluetooth** portables (Peripage, Munbyn, Xprinter…).

- Taille : 58mm de large, hauteur variable
- Une étiquette par feuille thermique
- Impression rapide sans encre

**Idéale pour :** étiqueter la marchandise à la réception, au dépôt.

### Feuille A4 — Avery L7159

Pour les **imprimantes de bureau classiques** (laser ou jet d'encre).

- Taille de la page : A4
- 24 étiquettes par page (3 colonnes × 8 rangées)
- Compatible planches autocollantes Avery L7159 et équivalents

**Idéale pour :** étiqueter en grande quantité, ou si vous n'avez pas d'imprimante thermique.

## Imprimer les étiquettes d'un produit

### Cas typique : vous venez de recevoir de la marchandise

1. Allez dans **Catalogue > Produits**
2. Cliquez sur le produit reçu
3. Cliquez sur **Imprimer des étiquettes**
4. Choisissez :
   - **Format** : Thermique 58mm ou Feuille A4
   - **Nombre de copies** : la quantité reçue (ex : 30 pièces → 30 étiquettes)
   - **Afficher le prix** : oui ou non selon votre besoin
5. Cliquez sur **Générer les étiquettes**
6. Une page s'ouvre avec les étiquettes — cliquez sur **🖨 Imprimer**

> **Astuce :** imprimez exactement autant d'étiquettes que d'articles reçus, et collez-les immédiatement à la réception.

### Étiquettes pour une variante spécifique

Si votre produit a des variantes (ex : `Boubou Rouge/L`) :

1. Ouvrez le produit
2. Dans la section Variantes, cliquez sur la variante concernée
3. Cliquez sur **Imprimer des étiquettes**
4. Choisissez le nombre de copies correspondant au stock de cette taille/couleur

L'étiquette affichera le SKU de la variante (`VET-0001-V1`) et ses attributs (`Rouge / L`).

---

## Impression par lot (toute une livraison)

Quand vous recevez une livraison avec plusieurs produits différents, vous pouvez tout imprimer en une seule opération.

1. Allez dans **Stock > Réceptions** (ou **Catalogue > Impression en lot**)
2. Cliquez sur **Imprimer étiquettes de livraison**
3. Pour chaque produit de la livraison, indiquez la quantité reçue
4. Choisissez le format (thermique ou A4)
5. Cliquez sur **Générer tout**
6. Toutes les étiquettes s'affichent dans une seule page — imprimez

## Paramètres d'impression

| Option | Description |
|--------|-------------|
| **Afficher le prix** | Décochez si vous ne souhaitez pas que le prix soit visible sur l'étiquette (ex : boutique avec prix négociés) |
| **Afficher le QR code** | Décochez si vous utilisez uniquement le code-barres pour votre douchette |

---

## Conseils pratiques

- **Imprimante thermique** : préférez le format thermique 58mm pour la mobilité — vous pouvez étiqueter directement au magasin ou à l'entrepôt avec une imprimante posée sur une table.
- **Grandes quantités** : utilisez le format A4 — une feuille = 24 étiquettes, c'est plus rapide que d'imprimer thermique une par une.
- **Badges PROMO** : si vous avez entré un prix barré supérieur au prix actuel, l'étiquette affiche automatiquement un badge **PROMO** rouge.
- **Coller au bon endroit** : collez l'étiquette sur l'emballage ou l'article de façon visible, en évitant les zones de pliage.
